Inheritance Types

This notebook discusses different types of inheritance.

  1. Single inehritance refers to the case when a subclass inherits from a single superclass
  2. Multiple inhertiance refers to the case where a subclass inherits from two or more classes.
  3. Multilevel inheritance refers to the case where the same class is both a parent and a child
  4. Hierarchical inheritance refers to the case where two or more classes are the subclasses of the same parent class
